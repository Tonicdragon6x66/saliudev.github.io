Alien Rescue

Alien Rescue (AR) is an online problem-based 3D immersive learning environment for sixth grade science, which is created by using React + Firebase + Blender. It combines best practices from educational research with innovative technologies to deliver an engaging learning experience.

I joined Alien Rescue (AR) team as a researcher, programmer, designer since Spring 2013. Although AR is not open for general public, you can still want to check out the period table I created within AR.
